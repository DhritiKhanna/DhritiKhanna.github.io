Mentors: Anand Yeolekar, R. Venkatesh

I was assigned the task of optimization of verifying sequentialized concurrent programs. The work on sequentialization translated the parallel program into a sequential program by nondeterministically assigning timestamps to the read and write memory accesses and they encode the semantics of the memory model as constraints on these timestamps. Even though sequentializing a program improves the performance over model checking a parallel program, all the interleavings are explored and the time is not reduced drastically. My task was to come up with program invariants imposing which will cut down the number of interleavings.
